# win-photo-autofix-ahk

## Requirements

- [AutoHotkey](https://www.autohotkey.com/)

## Usage

1. Place `.jpg` or `.png` images in the specified directory first.
1. Run `autofix.ahk`.

## Note

Please edit `autofix.ahk` before running it.

```ahk
imageDir := "E:\___DUMMY___\__auto_fix__"
```
